HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to user
  • html creates the structure of a webpage and contains other language within (such as CSS and JavaScript)
  • Semantic elements describe the content used. Use semantic as often as possible
  • "link" creates a connection to an external file
  • rel attribute specifies the relationship between the current doc. and the linked document source
  • href attribute specifies the location URL of the external resource. The href attribute is using a relative path.

CSS

Three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.

  • Internal style sheet contain CSS rules for the webpage in the head section of the HTML file. The rules only apply to that page, and you can configure CSS classes and IDs, which are used to identify elements, to style multiple elements in the page code. Again, a single change to the CSS rule will apply to all tagged page elements.
  • inline CSS. Inline styles apply to a specific HTML tag, using a style attribute with a CSS rule to style a particular page element only.
  • the most common way to style a webpage is by using an external CSS style sheet. This is preferred in most cases because it allows developers to keep all their CSS rules in a separate file, which makes design changes easier.
  • box-shadow adds shadow effects around an element's frame. You can set multiple effects separated by commas. A box shadow is described by X and Y offsets relative to the element, blur and spread radius, and color. In our example, the X offset is 5px and defines the width or extent of the shadow on the X axis (or to the right of the box); the Y offset is 10px and defines the extent on the Y axis (or just below the box).

Separation of concerns is a widely used term in web development and refers to separating code into different files based on use. Declarations contain two important components: the CSS property we want to apply and the value of the property

CSS Box Model

  • A margin property indicates how much space we want around the outside of an element.
  • The padding property adds space around the content inside an element.
  • Margin, Border, Padding, Content (From outside to inside)

Git

Commands

  • pwd: Print working directory
  • cd: Change directory
  • mkdir: Make directory
  • ls: List files
  • cd ..: Move back one level
  • git remote -v: Confirms link between local and server
  • git clone: Clones repo to local computer
  • touch ____ (file name): Creates text file inside directory
  • git checkout-b branch-name: Creates a new branch and switches to it
  • git checkout feature/branch: Switches to specified branch
  • git status: Checks what branch we are currently working on
